The Power Duo

MCP = Connectivity

Connects Claude to external systems:

  • • Databases & APIs
  • • GitHub, Jira, Slack
  • • File systems & cloud storage
  • • Any external data source

Skills = Procedures

Teaches Claude what to do with data:

  • • Business logic & workflows
  • • Data analysis patterns
  • • Report generation rules
  • • Domain expertise
Combined: MCP fetches the data → Skills process it intelligently

Code Execution Pattern (98% Token Reduction)

❌ Old Way (150,000 tokens)

  • • Load ALL tool definitions upfront
  • • Pass ALL results through context
  • • Model sees everything
  • • Massive token burn

✓ New Way (2,000 tokens)

  • • ONE execute_code tool
  • • Tools as code APIs
  • • Data stays in sandbox
  • • Only results returned

How It Works

// Instead of 42 GitHub tools in context...
// ONE execute_code tool that runs TypeScript:

const result = await mcp.github.listPullRequests({
  owner: 'myorg',
  repo: 'myrepo',
  state: 'open'
});

// Filter & process in sandbox (not context!)
const urgent = result.filter(pr =>
  pr.labels.includes('urgent')
);

// Return only summary to model
return { count: urgent.length, titles: urgent.map(p => p.title) };

Async Operations (November 25 Feature)

Long-running tasks no longer block. Fire and check back later. 

// Start long-running analysis
const taskId = await mcp.analytics.startAnalysis({
  dataset: 'sales_2024',
  type: 'comprehensive'
});

// Continue with other work...
await processOtherTasks();

// Check back for results
const status = await mcp.analytics.checkStatus(taskId);
if (status.complete) {
  const results = await mcp.analytics.getResults(taskId);
}

Progressive Tool Discovery

Tools presented as filesystem. Load only what you need. 

// Browse available servers (no connection needed!)
const servers = await mcp.discover();
// Returns: ['github', 'postgres', 'slack', 'jira']

// Read specific tool definition only when needed
const tool = await mcp.loadTool('github/createPullRequest');

// Or explore like a filesystem
const githubTools = await mcp.list('github/');
// Returns: ['issues/', 'pulls/', 'repos/', 'actions/']

.well-known Discovery

Servers advertise capabilities at:

https://server.com/.well-known/mcp.json

Browse before connecting. No token cost for discovery.
